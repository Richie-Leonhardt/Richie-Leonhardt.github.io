8月事件及反思

学习上：考研暑假复习的第二个月

  • 时间管理
    • 有时候学习效率比较低
    • 很多不想学习的情况
  • 英语
    • 单词已经背了4500，唯一值得庆幸的是背单词的习惯已经根深蒂固
    • 阅读已经有一定的感觉了，记住做阅读的时候一定要全神贯注
    • 接触了一点作文的写作
    • 应该每天再去接触一点长难句，翻译之类的
  • 政治
    • 再有一两天就能复习完近代史了
    • 实际上最好的方法是用核心考案听课做1000题，核心考案中没有的再去精讲精练中去补充
    • 多选题没有找到做题的立场所以错了很多。
    • 书居然看不进去，想想办法，实在不行就直接把重要的命题点直接抄下来也行。
  • C语言
    • 一本书刷完了
    • 做了部分真题，没有特别难的，但是细节陷阱特别多
  • Web
    • 开始进行第二遍到第五章JavaScript与HTML了，做完了后面的习题和编程练习
    • 光看书比较枯燥，还是自己动手写代码最快，要找到适合自己的学习方式
  • 数据结构
    • 第一遍已经复习完毕
    • 留下来的是主要一些比较难的编程题
    • 可以做真题了，到学校先做真题，遇到问题比较多的话可以再做严书的配套习题

生活

  • 基本已经能控制白天睡觉的问题，就按照之前的做，很有效

存在的问题

  1. 过于虚浮，有点swell了。
    • 沉下心来，要在考试中发挥100%的实力，就要学习150%
    • 不要轻视任何一个错误，做题的时候不会的不确定的一定要标注出来
    • 我的对手不是周围的人，而是跟我有同样的基础的那些看不见的对手
    • 要意识到现在我的英语政治依然很菜的，尤其英语六级360的耻辱
  2. 看书的时候总会想到其他一些无关紧要的东西，做题的时候这种情况比较少
    • 集中，集中，集中
    • 不要再想这些无关紧要的了，扇自己一巴掌，或者inhale一下
  3. 有时候吃饭吃太饱影响学习，血液都往消化系统流了脑子就不够清醒了
  4. 睡太晚会影响第二天的学习，应该规律好作息最好每天11点开始过一遍单词然后11：30睡觉

Don’t stagnate there.

用你的时间，去做有意义的事情。
